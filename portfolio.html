Project Two

SKILLS: Python, Excel, Jupyter Notebook, SQL, Web Scraping, Twitter API, Data Cleaning, Communication, Presentation

Introduction

WeRateDogs (@dog_rates) is a Twitter account that gives ratings to people's dogs along with a lighthearted comment. The denominator of these scores is almost always 10. However, the numerators? Frequently more than 10. 13/10, 14/10, 15/10, etc. Over 4 million people follow WeRateDogs, and it has been featured in international media. In this report, I will reveal three insights about WeRateDogs from the WeRateDogs tweet archive dataset.

Procedure: Data Analysis and Presentation Workflow

  • Used Twitter API to gather data
  • Scraped the web using Beautiful Soup python package to gather more data
  • Harvested more data from database using SQL
  • Viewed the datasets in Excel as part of data assessment
  • Imported datasets into Jupyter Notebook
  • Cleaned datasets and combine all into a single dataset
  • Started exploratory analyzes on the dataset
  • Created explanatory visualizations and communicated findings to stakeholders

Insights

  • The source that is mostly used for making a tweet by WeRateDogs
  • Does the number of likes likely influence the number of followers WeRateDogs get
  • Does the number of re-tweets likely influence the number of likes that WeRateDogs
